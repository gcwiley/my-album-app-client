## About this App

'My Album App' is a simple web application that allows users to manage their favorite music albums using a clean and easy to understand user interface (UI). 

This web application is a basic CRUD (Create, Read, Update, Delete) web application that uses the MERN Stack. 

I am teaching myself Full Stack web development so I plan to use this web-application as a testing development evironment. 

I plan to add additional code over time. Please let me know if you have any questions.
